## 📝 技术说明

### 使用的 UE5 特性

- **Enhanced Input System** - 新一代输入系统
- **Gameplay Ability System** (预留) - 技能系统框架
- **Niagara** (预留) - 粒子特效
- **MetaSounds** (可选) - 音频系统

### 代码规范

- 遵循 Epic Games C++ 编码标准
- 使用智能指针和 UE4/UE5 内存管理
- 蓝图可调用函数使用 `UFUNCTION(BlueprintCallable)`
- 属性使用 `UPROPERTY` 宏标记
